##克隆项目：

你先得做一些准备工作，你可能需要先安装git命令行工具；
> 推荐的git客户端：[Git][2]、[GitHub][3] 或者其他git客户端；

打开你的cmd命令窗口（ window键 + r 并输入cmd 回车）;
```cmd
git clone https://github.com/RUSHUI/imooc-video.git
git checkout First
```
